# AI Closet Stylist
옷 정보를 등록해두면, 날씨에 맞춰 AI가 오늘의 코디를 추천해주는 웹사이트


# 프로젝트 기능 소개

옷 등록	: 사용자가 자신의 옷 사진, 이름, 색상, 종류(상의/하의/신발 등)를 등록
날씨 연동	: OpenWeatherMap API로 현재 날씨/기온 받아오기
AI 코디 추천	 : Gemini 1.5 Flash가 날씨와 옷 데이터를 분석해 여러 코디 조합 추천
색상/스타일 고려 :	AI가 색조합·톤 밸런스를 반영해 코디 생성
코디 결과 표시 :	React 프론트엔드에서 코디별 설명, 추천 이유 표시
데이터 저장 :	SQLite나 JSON 파일로 간단히 저장 가능
